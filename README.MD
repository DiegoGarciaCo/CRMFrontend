# Realtor CRM - Next.js Frontend

A modern CRM frontend for real estate agents, built with Next.js 20, TypeScript, and TailwindCSS.

## Features
- Agent dashboard with property and client management
- Authentication via Google and Apple
- Stripe integration for payments
- Responsive and mobile-friendly UI
- Real-time updates (WebSockets)

## Getting Started

### Requirements
- Node.js 20+
- npm or yarn
- Backend API running

### Installation
```bash
git clone https://github.com/yourusername/realtor-crm-frontend.git
cd realtor-crm-frontend
npm install
